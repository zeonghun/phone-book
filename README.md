# 기능
● 유저는 연락처를 생성할 수 있다  
● 연락처에는 이름과 전화번호 정보가 필요하다  
● 유저는 연락처 리스트를 볼 수 있다  
● 유저는 이름으로 연락처를 검색할 수 있다  

# 사용 기술
● HTML  
● CSS  
● Javascript  
● React  
● Redux
